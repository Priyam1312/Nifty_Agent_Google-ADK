📊 Nifty 50 Data Analysis Agent

An intelligent data analysis agent for the Nifty 50 stock market, built using Google ADK, MCP server architecture, and Gemini API. The system processes Kaggle datasets to generate insights, visualizations, automate analysis workflows, and support data-driven decisions.

⸻

🚀 Features
	•	📈 Nifty 50 market data analysis
  •	📊 Automatic plots & graph generation
	•	🤖 Gemini-powered intelligent insights
	•	🔌 MCP-based tool integration
	•	🗃️ Database-backed data handling
	•	⚡ Scalable and modular architecture

⸻

🛠️ Tech Stack
	•	Google ADK
	•	MCP Server
	•	Python (MatPlotLib)
	•	Gemini API
	•	Kaggle Datasets
	•	Database (SQL/NoSQL)

⸻

Create the .env file and add your_api_key in the second line

GOOGLE_GENAI_USE_VERTEXAI=FALSE
GOOGLE_API_KEY=your_api_key_here
